# S.E.E.D. 요원 신상 카드 생성기

카미가카리 RPG 캠페인 「Case Number Six」용 요원 ID 카드 생성기입니다.

## 배포 URL
> GitHub Pages 설정 후 자동으로 생성됩니다.

## 기능
- Document Mode / Anomaly Mode 선택
- 요원 정보 입력 (이름, ID, 계급, 소속, 이메일, 관리등급, 전문능력, 초상여부)
- 요원 사진 업로드
- 명함 사이즈 PNG 출력 (1440×864px)

## 사용 기술
- 순수 HTML / CSS / JS (의존성 없음)
- Google Fonts (Noto Sans KR, IBM Plex Mono, Noto Serif KR, Black Han Sans)
- html2canvas 1.4.1
